Introduction & Interactive Simulations

uSort-M is a method for generating parsed protein libraries from oligonucleotide pool inputs. Use the interactive simulations below to explore how different library parameters affect variant distributions and recovery rates.

Library Distribution Simulator

Explore how different parameters affect the distribution of variant counts in your protein library.

Recovery Rate Simulator

Simulate how sample size affects the recovery rate of variants from your library.

Software Installation

uSort-M provides tools for modeling and simulating protein library distributions and recovery rates in your own Python environment.

Installation

pip install -e .

Usage

from usort_m import simulations
